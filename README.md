# Reverse Integer - Resolução em Java

Exercício Reverse Integer - LeetCode, resolvido usando Java.

## Descrição do Problema

  Este projeto resolve o problema Reverse Integer, que consiste em inverter os dígitos de um número inteiro de 32 bits com sinal.

  O método "reverseInteger" receberá um inteiro "x", e deverá retornar o número com seus dígitos invertidos.
  
  Caso o número invertido ultrapasse o intervalo permitido para inteiros de 32 bits [-2³¹, 2³¹ - 1], o programa deve retornar 0.

  Também é assumido que o ambiente não permite armazenar inteiros de 64 bits.

## Exemplos

### Exemplo 1

  Entrada: x = 123
  
  Saída: 321

### Exemplo 2

  Entrada: x = -123
  
  Saída: -321

### Exemplo 3

  Entrada: x = 120
  
  Saída: 21
